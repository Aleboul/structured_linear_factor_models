# Github repository for the paper ``Structured linear factor models for tail dependence''


## Repository Description

This repository contains code and data for replicating experiments and analyses presented in the main paper. The repository is organized into two main directories:

**1. case studies**

In this directory, you'll find datasets and Python scripts for performing the analysis described in Section 5 of the paper. 

**2. simulation results**

This directory contains Python code for conducting numerical experiments outlined in Section 4 of the main paper.
